# Ahlam Cheryala Venue Portfolio

React + Vite portfolio for a wedding planner and venue manager.

## Run locally

```bash
npm run install-all
npm run dev
```

## Edit content

Update the main content in:

```text
client/src/portfolioData.js
```

Add photos in:

```text
client/src/assets
```

Add videos in:

```text
client/public/videos
```

## Deploy on Netlify

The project includes `netlify.toml` with the correct settings.
